O que é HTML?

É a linguagem base da web, usada para criar e estruturar páginas na internet.

Características:

  • Utiliza tags para marcar o conteúdo (como <p> para parágrafos).
  • Permite adicionar textos, imagens, vídeos e links
  • Trabalha em conjunto com CSS e JavaScript.
    •

    Exemplo de código:

        <h1>Olá, mundo! </h1>
    <p>Essa é uma página HTML.</p>
